Connect Google Analytics

Sign in with the Google account that has access to your GA4 property (kassbyjess@gmail.com)

One-time setup (2 minutes):
1. Go to console.cloud.google.com — sign in as kassbyjess@gmail.com
2. Create a new project (or select existing) → name it "Kass Analytics"
3. Go to APIs & Services → Library → search "Google Analytics Data API" → Enable
4. Go to APIs & Services → Credentials → Create Credentials → OAuth client ID
5. Application type: Web application
6. Add Authorized JavaScript origin: https://kasskreative.com
7. Also add http://localhost if testing locally
8. Copy the Client ID (looks like 123456-xxx.apps.googleusercontent.com) and paste it above

Note: If prompted to configure consent screen, choose "External", fill in app name "Kass Analytics", your email, and save. You only need to do this once.
